Time Comlexity: O(n)
Space Complexity: O(n)

Пример #1:
```
Input: nums = [1, 3, 5, 7, 9], k = 3
Output: [3.0, 5.0, 7.0]
```

Пример #2:
```
Input: nums = [10, 20, 30, 40, 50], k = 2
Output: [15.0, 25.0, 35.0, 45.0]
```

```
Input: nums = [5, 10, 15], k = 1
```
